## Start

```
cd $HOME
git clone git@github.com:fukajun/dotfiles.git
cd dotfiles

ln -nsf $HOME/dotfiles/.Xmodmap      $HOME/.Xmodmap
ln -nsf $HOME/dotfiles/.bash_profile $HOME/.bash_profile
ln -nsf $HOME/dotfiles/.bashrc       $HOME/.bashrc
ln -nsf $HOME/dotfiles/.commands     $HOME/.commands
ln -nsf $HOME/dotfiles/.inputrc      $HOME/.inputrc
ln -nsf $HOME/dotfiles/.profile      $HOME/.profile
ln -nsf $HOME/dotfiles/.screenrc     $HOME/.screenrc
ln -nsf $HOME/dotfiles/.tmux.conf    $HOME/.tmux.conf
ln -nsf $HOME/dotfiles/.vimrc        $HOME/.vimrc
ln -nsf $HOME/dotfiles/gitconfig     $HOME/.gitconfig
```
